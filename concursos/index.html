    Bem Vindo ao MkDocs

    For full documentation visit mkdocs.org.

    Comandos Básicos

    • mkdocs new [dir-name] - Cria um novo projeto.
    • mkdocs serve - Dá início ao MkDocs no navegador.
    • mkdocs build - Constrói a documentação do site.
    • mkdocs help - Exibe a tela de ajuda.
    • CTRL C - Fecha MkDocs.
